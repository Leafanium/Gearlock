Gearlock - Unity Version: 2022.30.60f1 -
This game is a very simple first person shooter game where the main goal is to survive as long as possible by killing all of the incoming robots. It is red necked themed and it also has a theme of gears hence the name Gear lock. There is a health system and the robots will chase you and if they reach you they will slowly take away your health. The only way to heal your self back up is to run over the little bandages that are on the floor with the white glow. We also have a timer that will show how long you are in the game without it ending. The only thing that we are missing is the ending condition. There technically is not a losing or wining part of the game because we were having problems with making it work. There is only one gun and it has 6 shells and you can reload it by pressing "r". The music that is playing is from COD Black ops 2 zombies and also the sound effects from the gun and the bandage are from World of Warcraft. I also am not too sure where the rest of the assets are from but I am sure they are from the unity shop for free. We recycled a lot of the scripts from the main homework that we had before these projects.
